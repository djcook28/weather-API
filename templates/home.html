Weather Data API

Resting API for getting weather data by station number

URL format for a specific station and date: http://127.0.0.1:5000/api/v1/station/date
Example: http://127.0.0.1:5000/api/v1/10/19881025

URL format for full year date for a specific station: http://127.0.0.1:5000/api/v1/station/year
Example: http://127.0.0.1:5000/api/v1/10/1988

URL format for all station data: http://127.0.0.1:5000/api/v1/station
Example: http://127.0.0.1:5000/api/v1/10
